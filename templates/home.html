Welcome to Sketch Map Analyzer (SMA)

The SMA is a tool that analysis the accuracy and completeness of the drawn spatial information in the input Sketch Map with respect to spatial information in the Geo-referenced Map


The accuracy of input map is measured based on the preserved Qualitative Spatial Configurations:

  • Topological relations between polygonal features: i.e. Connected, Disconnected, Overlaps, etc.
  • Topological relations between linear features: i.e. Connected, Disconnected, crosses, etc.
  • LeftRight relations between polygonal and linear features: i.e. Left_of, Right_of, Crosses, etc.
  • Linear ordering relations between polygonal along a route: i.e. Before, After, Overlaps, etc.
  • Topological relations between polygonal and linear features: i.e. Connected, Disconnected, crosses, etc.
  • Relative orientation relations between linear features: i.e. Left_of, Right_of; Front_of, Half_left, etc.
